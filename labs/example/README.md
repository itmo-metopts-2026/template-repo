# Пример — Линейная регрессия методом градиентного спуска

> **Это образцовый пример оформления лабораторной работы.**
> Используйте его как шаблон при выполнении lab1–lab4.

## Структура

| Файл | Назначение |
|------|-----------|
| `model.py` | Реализация алгоритма (класс + метрика) |
| `research.ipynb` | Jupyter Notebook с экспериментами |
| `report.md` | Отчёт по лабораторной работе |
| `results/figures/` | Сохранённые графики |

## Запуск

```bash
# Из корня репозитория
make install
make jupyter
```

Затем откройте `labs/example/research.ipynb` в JupyterLab.

## Воспроизводимость

Все эксперименты используют фиксированный `seed = 42`.
Для воспроизведения результатов выполните все ячейки notebook
последовательно сверху вниз.

## Как сдать лабораторную

Подробная инструкция — в [главном README](../../README.md#как-сдать-лабораторную-работу). Кратко:

1. Убедитесь, что в папке лабы есть `model.py`, `research.ipynb`, `report.md`, графики в `results/figures/`.
2. Из корня репозитория: `make lint` (при ошибках — `make format`).
3. Закоммитьте и запушьте:
   ```bash
   git add labs/labN/    # N — номер вашей лабы
   git commit -m "Лабораторная работа N"
   git push origin main
   ```
4. Отправьте преподавателю ссылку на репозиторий или на Merge/Pull Request (если сдача через ветку).
