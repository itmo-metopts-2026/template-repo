# Отчёт — Пример лабораторной работы

## Линейная регрессия методом градиентного спуска

---

## 1. Цель работы

Реализовать алгоритм линейной регрессии, обучаемой методом градиентного спуска,
без использования готовых библиотек машинного обучения. Исследовать поведение
алгоритма на синтетических данных и оценить качество аппроксимации.

## 2. Постановка задачи

Дана выборка из *n* объектов, каждый из которых описывается *m* числовыми
признаками. Требуется:

1. Построить линейную модель вида **ŷ = Xw + b**, минимизирующую
   среднеквадратичную ошибку (MSE) на обучающей выборке.
2. Реализовать обучение модели методом градиентного спуска.
3. Визуализировать результаты: аппроксимирующую прямую и динамику функции потерь.

## 3. Теоретические основы

### 3.1. Линейная регрессия

Линейная регрессия предполагает линейную зависимость целевой переменной *y*
от вектора признаков *x*:

> ŷ = x₁w₁ + x₂w₂ + … + xₘwₘ + b

где **w** = (w₁, …, wₘ) — вектор весов, **b** — свободный член (bias).

### 3.2. Функция потерь

В качестве функции потерь используется **среднеквадратичная ошибка** (MSE):

> L(w, b) = (1/n) · Σᵢ₌₁ⁿ (yᵢ − ŷᵢ)²

MSE штрафует за большие отклонения квадратично, что делает её чувствительной
к выбросам, но при этом гладкой и дифференцируемой — удобной для оптимизации.

### 3.3. Градиентный спуск

Градиентный спуск — итерационный алгоритм оптимизации. На каждом шаге параметры
обновляются в направлении антиградиента:

> w ← w − η · ∂L/∂w
> b ← b − η · ∂L/∂b

где **η** — скорость обучения (learning rate).

Градиенты для MSE:

> ∂L/∂w = (2/n) · Xᵀ · (ŷ − y)
> ∂L/∂b = (2/n) · Σᵢ (ŷᵢ − yᵢ)

### 3.4. Сходимость

При достаточно малом значении η и выпуклости функции потерь (что гарантируется
для MSE с линейной моделью) градиентный спуск сходится к глобальному минимуму.

## 4. Описание реализации

Реализация выполнена на языке Python без использования sklearn.

### 4.1. Модуль модели

Файл: [`labs/example/model.py`](model.py)

Содержит:

- **Класс `LinearRegressionGD`** — модель линейной регрессии с обучением
  градиентным спуском. Основные методы:
  - `fit(X, y)` — обучение модели на обучающей выборке;
  - `predict(X)` — вычисление предсказаний для новых данных.

- **Функция `mse(y_true, y_pred)`** — вычисление среднеквадратичной ошибки.

Реализация включает:
- проверку типов и размерностей входных данных;
- автоматическое преобразование одномерного X в матрицу-столбец;
- сохранение истории значений функции потерь по эпохам.

### 4.2. Утилита фиксации случайности

Файл: [`labs/utils/seed.py`](../utils/seed.py)

Функция `set_global_seed(seed)` фиксирует состояние генераторов случайных
чисел: `random`, `numpy.random` и переменную `PYTHONHASHSEED`.

## 5. Экспериментальная часть

Полный процесс исследования представлен в Jupyter Notebook:
[`labs/example/research.ipynb`](research.ipynb)

### 5.1. Данные

Были сгенерированы синтетические данные (100 объектов) вида:

> y = 3x + 7 + ε

где ε ~ N(0, σ²) — нормальный шум.

### 5.2. Гиперпараметры

| Параметр | Значение |
|----------|----------|
| Скорость обучения (η) | 0.01 |
| Количество итераций | 1000 |
| Размер выборки | 100 |
| Seed | 42 |

### 5.3. Ход эксперимента

1. Фиксация случайности (`seed = 42`).
2. Генерация синтетических данных.
3. Обучение модели `LinearRegressionGD`.
4. Вычисление MSE на обучающей выборке.
5. Построение графиков: аппроксимирующая прямая и кривая обучения.
6. Сохранение графиков в `results/figures/`.

## 6. Результаты

### 6.1. Качество модели

Модель успешно аппроксимировала линейную зависимость.
Значение MSE на обучающей выборке стремится к дисперсии шума,
что подтверждает корректность реализации.

### 6.2. Визуализация

Графики сохранены в директории: [`results/figures/`](results/figures/)

- **regression_fit.png** — аппроксимирующая прямая на фоне данных.
- **loss_curve.png** — динамика функции потерь (MSE) по эпохам.

Кривая обучения демонстрирует монотонное убывание MSE, что свидетельствует
о корректной работе градиентного спуска.

2. На синтетических данных модель корректно восстанавливает линейную зависимость.
