# Лабораторная работа №1

## Структура

| Файл | Назначение |
|------|-----------|
| `model.py` | Реализация алгоритма |
| `research.ipynb` | Jupyter Notebook с экспериментами |
| `report.md` | Отчёт по лабораторной работе |
| `results/figures/` | Сохранённые графики |

## Как начать

1. Изучите пример оформления в [`labs/example/`](../example/)
2. Реализуйте алгоритм в `model.py`
3. Проведите эксперименты в `research.ipynb`
4. Напишите отчёт в `report.md`

## Запуск

```bash
make install
make jupyter
```

## Воспроизводимость

Не забудьте зафиксировать `seed = 42` в начале notebook.

## Как сдать лабораторную

Подробная инструкция — в [главном README](../../README.md#как-сдать-лабораторную-работу). Кратко:

1. Убедитесь, что в папке есть `model.py`, `research.ipynb`, заполненный `report.md`, графики в `results/figures/`.
2. Из корня репозитория: `make lint` (при ошибках — `make format`).
3. Закоммитьте и запушьте:
   ```bash
   git add labs/lab1/
   git commit -m "Лабораторная работа 1"
   git push origin main
   ```
4. Отправьте преподавателю ссылку на репозиторий или на Merge/Pull Request (если сдача через ветку).
